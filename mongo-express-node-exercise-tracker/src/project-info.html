DETAILS

  • Submit a valid URL in the form above.
  • For a valid url the api returns a JSON with the original url and the shortened URL:

    • {"original_url": "http://www.exampleUrl.com" ,"short_url": "root_url/a/addlurl"}

  • Clicking on the url link will redirect you to the same website as the original link
  • Providing an invalid URL will return the following JSON:

    • {"error": "invalid URL" }

    EXAMPLE (click on the link)

  • Sample Input

    • http://www.google.com

  • Sample JSON Output

    • {"original_url": "http://www.google.com" ,"short_url": "root_url/a/1"}

  • Sample Working Link
    • [root_url]/a/1
