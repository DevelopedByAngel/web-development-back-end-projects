DETAILS

  • Submit a valid URL in the form above.
  • For a valid url the api returns a JSON with the original url and the shortened URL:

    • {"original_url": "www.exampleUrl.com" ,"short_url": "root_url/api/shorturl/ex"}

  • Clicking on the url link will redirect you to the same website as the original link
  • Providing an invalid URL will return the following JSON:

    • {"error": "invalid URL" }

    EXAMPLES (click on the links)

  • Date Input
    • [root_url]/api/timestamp/2018-11-20

    JSON output: {"unix":1542672000000,"utc":"Tue, 20 Nov 2018 00:00:00 GMT"}

  • UNIX Input
    • [root_url]/api/timestamp/1545264000000

    JSON output: {"unix":1545264000000,"utc":"Thu, 20 Dec 2018 00:00:00 GMT"}

  • Invalid Date Input
    • [root_url]/api/timestamp/alskfy(87

    JSON output: {"unix":null,"utc":"Invalid Date"}

  • No date Input
    • [root_url]/api/timestamp/

    JSON output: current date generated by new Date();
