Express & Node.js Metric - Imperial Converter Microservice

TRY IT OUT

Insert measurement here with abbreviated unit

DETAILS

  • Input valid measurement with abbreviated unit (metric or imperial) to converted. The following are allowed:

    l to gal, kg to lbs, km to mi vice versa
  • This project returns a JSON converting a unit from metric to imperial vice versa
  • For a valid measurement input, the api returns a JSON having the structure:

    • {"initNum": <number input> ,"initUnit": <unit inputted>, "returnNum": <number converted> , "returnUnit": <unit converted>, "string": <information>}

  • If only the unit is provided the system will assume 1 as the measurement. (e.g. km = 1 km)

    • EXAMPLES (click on the links)

  • Liters to Gallons

    • {"initNum": 1 ,"initUnit": "l", "returnNum": 0.2642, "returnUnit": "gal", "string": "1 L is 0.2642 gal"}

  • Kilograms to Pounds

    • {"initNum": 1 ,"initUnit": "kg", "returnNum": 2.2046, "returnUnit": "lbs", "string": "1 kg is 2.2046 lbs"}

  • Kilometer to Miles

    • {"initNum": 1 ,"initUnit": "km", "returnNum": 0.6214, "returnUnit": "mi", "string": "1 km is 0.6214 mi"}
